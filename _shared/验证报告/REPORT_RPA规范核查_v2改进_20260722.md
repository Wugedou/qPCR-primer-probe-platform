## 2. v2 改进措施

### 2.1 产物大小收紧 (100-500 → 80-400)
- 新增 `product_size_opt_min/max` (100-200bp 最优区间)
- 避免长 amplicon 降低扩增效率

### 2.2 长串单核苷酸检查
- 新增 `_max_homopolymer_run()` 函数
- 阈值: >5nt 连续同碱基 = warning (Daher 2016 规范)

### 2.3 小重复序列检查
- 新增 `_has_long_repeats()` 函数
- 检测 3+ 重复单元 × 3+ 次

### 2.4 exo-probe 3' 端封闭标注
- 在 note 中明确: "3' 端加磷酸基团(PO4)或双脱氧核苷酸(ddC)封闭"

### 2.5 warning/issue 分离
- 长串单核苷酸和小重复标为 warning (非 fatal)
- 不影响 `accepted` 判定

## 4. 未实施的改进 (后续工作)

1. **nfo 探针** — 5'FAM+THF, 配套 5'Biotin 反向引物, 侧流层析
2. **fpg 探针** — 32-35bp, 5'淬灭+dR-荧光, 灵敏度低于 exo
3. **Tm 约束去除** — 当前 Tm 窗口 35-75°C 可保留作信息性输出, 但不作为约束
4. **5' 加性** — TwistDX 建议 5' 加 10-20nt 无关序列提高重组酶负载 (需实验验证)
